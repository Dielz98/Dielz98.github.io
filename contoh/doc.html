Cara Olah Data

Ini adalah section terpenting yang menjelaskan bagaimana cara pengolahan datanya. Kami menggunakan video tutorial. Anda cukup ikuti saja videonya. Anda akan mendapatkan hasil yang serupa dengan yang kami kerjakan.

Olah Data OxMetrics

Pengolahan data di OxMetrics ditujukan untuk mencari Covariance, Value at Risk, Conditional Variance dan Conditional Correlation.

Data input yang dipergunakan pada proses pengolahan data covar, mes, dan panel data didapat dari perhitungan menggunakan OxMetrics

Data input untuk pengolahan data di OxMetrics adalah data-data excel dengan tulisan fix

Data output contohnya INDONESIA.in7 adalah data output Indonesia dari olahdata OxMetrics. INDONESIA.xlsx adalah data output Indonesia dari olahdata OxMetrics dalam format excel

Tonton video berikut yang menjelaskan cara olah data di OxMetrics. Ikuti langkah-langkahnya untuk mendapatkan hasil yang serupa

Regresi Panel Data

Data CoVaR hasil perhitungan di Olah Data Covar adalah data daily. Sedangkan data yang akan diregresikan dalam format monthly. Pada sesi ini dijelaskan bagaimana mengkonversikan data high-frequency (harian) menjadi data low-frquency (bulanan)

Data Leverage, GDP, Inflation, dan Size pada variabel dependen adalah data tahunan (yearly). Pada sesi ini juga akan dijelaskan bagaimana mengkonversikan data low-frquency (tahunan) menjadi data high-frequency (bulanan)
